## [1.0.0] - 2026-02-02

初回リリース。スライドを JSON データで定義してブラウザに表示するプレゼンテーションツールです。

### 追加

- **5 種類のレイアウト** — `layout` フィールドで `center` / `content` / `two-column` / `bleed` / `custom` を切り替え
- **テーマのカスタマイズ** — slides.json の `theme` フィールド（色・フォント・カスタム CSS）でカスタマイズ、または
  `theme-colors.json` で色を上書き
    - カスタムフォントの読み込み（ローカルファイル / Google Fonts）
    - `baseFontSize` によるフォントサイズの一括スケール
    - すべての色を CSS 変数として適用（`rgba()` 対応の `-rgb` バリアント付き）
- **音声ガイド & 自動送り** — スライドごとに音声ファイルを割り当て
    - 手動再生（スピーカーアイコンで再生 / 停止）
    - 自動再生（スライド遷移時に音声を自動再生）
    - 自動スライドショー（音声終了時に次のスライドへ自動送り）
- **発表者ビュー** — 専用の発表者ウィンドウを開く
    - スピーカーノート、要点サマリー、次 / 前スライドのプレビュー
    - BroadcastChannel によるメインウィンドウとの双方向リアルタイム同期
    - キーボードナビゲーション（矢印キー / スペース）
    - 発表者ビューからの音声コントロール
- **国際化 (i18n)** — 日本語（`ja-JP`）/ 英語（`en-US`）/ フランス語（`fr-FR`）
    - ブラウザの言語設定からの自動検出
    - 設定ウィンドウからの手動切り替え（`localStorage` に保存）
- **アドオンシステム** — カスタムコンポーネントをアドオンとして追加・登録
- **スライドパッケージ** — スライド内容を npm パッケージとしてエクスポート・インポート・配布
    - `npm run export:slides` で `.tgz` パッケージを生成（参照アセットを自動検出して同梱）
    - `VITE_SLIDE_PACKAGE` 環境変数でインポート（ローカルパス / npm パッケージ）
- **ロゴ設定** — `meta.logo` フィールドでプレゼンテーションのロゴをカスタマイズ
- **スライドメタ** — トランジション、背景色、背景画像を制御
